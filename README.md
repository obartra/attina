# Attina 🐟

Mermaid react renderer compatible with React 15.5+ based off [React Mermaid](https://github.com/jasonbellamy/react-mermaid)

Uses [Mermaid](https://mermaidjs.github.io/mermaidAPI.html) under the hood

## Setup

Install with:

```shell
$ yarn add attina
```
